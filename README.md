# Uart.py 
Attach GPS Module as follows
 - 3.3V power   -> pin 37 (3v3_en)
 - Grnd         -> pin 38 (Gnd) 
 - TX           -> pin 12 (GP9)
 - RX           -> pin 11 (GP8)
